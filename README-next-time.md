# RM Refresher Website

This is the editable project package for the RM Refresher website.

## Run Locally

From this folder, run:

```sh
./bin/next-run dev
```

Then open:

```txt
http://localhost:3000
```

## Main Edit Locations

- `app/` - website pages
- `components/` - reusable page sections
- `data/products.ts` - products, benefits, lifestyle cards, brew steps
- `public/images/` - website images

This package keeps `node_modules` so it can be reopened and edited without reinstalling dependencies.
